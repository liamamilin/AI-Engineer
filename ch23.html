28  Chapter 23 — Building Retrieval-Augmented Systems

28.1 Introduction

Retrieval-Augmented Generation (RAG) represents one of the most powerful architectural patterns in modern AI systems. By combining language models with external knowledge bases, RAG systems can answer questions grounded in specific documents, reason about current information, and maintain factual accuracy beyond what models have learned during training.

Chapter 9 introduced pipelines as the structural backbone of AI systems. RAG demonstrates the practical application of pipeline architecture: retrieval transforms raw queries into context-rich prompts, which then guide model inference toward accurate, grounded responses.

This chapter provides a comprehensive guide to building production-ready RAG systems, covering document processing, embedding strategies, vector databases, retrieval optimization, and evaluation.

28.2 The RAG Architecture

28.2.1 Pipeline Overview

A complete RAG system follows this pipeline:

user query
    ↓
query preprocessing
    ↓
query embedding
    ↓
vector search (vector database)
    ↓
context selection and ranking
    ↓
prompt construction (retrieved context + query)
    ↓
model inference
    ↓
answer generation
    ↓
answer validation
    ↓
response

Each stage contributes to the final quality and reliability of the system.

28.2.2 Component Responsibilities

Document Processing: Ingest, clean, and structure documents Embedding Generation: Convert text to vector representations Vector Storage: Store and index embeddings efficiently Retrieval: Find relevant documents based on query similarity Context Construction: Assemble retrieved information into prompts Answer Generation: Generate responses using retrieved context Validation: Verify answer quality and relevance

28.3 Step 1: Document Processing

28.3.1 Ingestion Pipeline

Create a robust document ingestion system:

# src/document_processor.py
from typing import List, Dict
import os
from pathlib import Path

class DocumentProcessor:
    def __init__(self):
        self.supported_formats = ['.txt', '.md', '.pdf', '.docx']
    
    def ingest_directory(self, directory: str) -> List[Dict]:
        """Ingest all documents from a directory."""
        documents = []
        
        for file_path in Path(directory).rglob('*'):
            if file_path.suffix in self.supported_formats:
                doc = self.process_file(str(file_path))
                if doc:
                    documents.append(doc)
        
        return documents
    
    def process_file(self, file_path: str) -> Dict:
        """Process a single document."""
        try:
            with open(file_path, 'r', encoding='utf-8') as f:
                text = f.read()
            
            # Extract metadata
            metadata = self.extract_metadata(file_path, text)
            
            # Clean and preprocess
            cleaned_text = self.clean_text(text)
            
            return {
                'id': self.generate_id(file_path),
                'content': cleaned_text,
                'metadata': metadata,
                'source': file_path
            }
        except Exception as e:
            print(f"Error processing {file_path}: {e}")
            return None
    
    def clean_text(self, text: str) -> str:
        """Clean and normalize text."""
        # Remove extra whitespace
        text = ' '.join(text.split())
        
        # Normalize line breaks
        text = text.replace('\r\n', '\n').replace('\r', '\n')
        
        return text.strip()
    
    def extract_metadata(self, file_path: str, text: str) -> Dict:
        """Extract metadata from document."""
        filename = os.path.basename(file_path)
        file_size = os.path.getsize(file_path)
        
        return {
            'filename': filename,
            'file_size': file_size,
            'file_path': file_path,
            'word_count': len(text.split()),
            'char_count': len(text)
        }
    
    def generate_id(self, file_path: str) -> str:
        """Generate a unique ID for the document."""
        return str(abs(hash(file_path)))

28.3.2 Text Chunking

Large documents must be split into manageable chunks for effective retrieval:

# src/chunker.py
from typing import List
import re

class TextChunker:
    def __init__(self, chunk_size=500, chunk_overlap=50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
    
    def chunk_document(self, document: Dict) -> List[Dict]:
        """Split a document into chunks."""
        text = document['content']
        chunks = self.split_text(text)
        
        chunked_docs = []
        for i, chunk_text in enumerate(chunks):
            chunk = {
                'id': f"{document['id']}_chunk_{i}",
                'content': chunk_text,
                'metadata': {
                    **document['metadata'],
                    'chunk_index': i,
                    'total_chunks': len(chunks),
                    'document_id': document['id']
                },
                'source': document['source']
            }
            chunked_docs.append(chunk)
        
        return chunked_docs
    
    def split_text(self, text: str) -> List[str]:
        """Split text into chunks with overlap."""
        chunks = []
        start = 0
        
        while start < len(text):
            end = start + self.chunk_size
            
            # Try to break at a sentence boundary
            if end < len(text):
                # Find the last sentence ending
                last_period = text.rfind('.', start, end)
                last_question = text.rfind('?', start, end)
                last_exclamation = text.rfind('!', start, end)
                
                # Use the furthest sentence boundary
                boundary = max(last_period, last_question, last_exclamation)
                
                if boundary > start:
                    end = boundary + 1  # Include the punctuation
            
            chunk = text[start:end].strip()
            if chunk:
                chunks.append(chunk)
            
            # Move start position with overlap
            start = end - self.chunk_overlap
        
        return chunks

28.3.3 Advanced Chunking Strategies

Different chunking strategies suit different use cases:

Semantic Chunking: Split at semantic boundaries using sentence embeddings

# src/semantic_chunker.py
from sentence_transformers import SentenceTransformer

class SemanticChunker:
    def __init__(self, similarity_threshold=0.8):
        self.model = SentenceTransformer('all-MiniLM-L6-v2')
        self.similarity_threshold = similarity_threshold
    
    def chunk_semantically(self, text: str) -> List[str]:
        """Split text based on semantic similarity."""
        sentences = self.split_into_sentences(text)
        
        if len(sentences) <= 1:
            return sentences
        
        # Encode sentences
        embeddings = self.model.encode(sentences)
        
        # Group sentences by semantic similarity
        chunks = []
        current_chunk = [sentences[0]]
        
        for i in range(1, len(sentences)):
            similarity = self.cosine_similarity(
                embeddings[i-1],
                embeddings[i]
            )
            
            if similarity < self.similarity_threshold:
                chunks.append(' '.join(current_chunk))
                current_chunk = [sentences[i]]
            else:
                current_chunk.append(sentences[i])
        
        if current_chunk:
            chunks.append(' '.join(current_chunk))
        
        return chunks
    
    def split_into_sentences(self, text: str) -> List[str]:
        """Split text into sentences."""
        return re.split(r'(?<=[.!?])\s+', text)
    
    def cosine_similarity(self, a, b):
        """Calculate cosine similarity between two vectors."""
        import numpy as np
        return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

28.4 Step 2: Embedding Generation

28.4.1 Embedding Service

# src/embedding_service.py
from openai import OpenAI
import os
from dotenv import load_dotenv

load_dotenv()

class EmbeddingService:
    def __init__(self, model="text-embedding-3-small"):
        self.client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))
        self.model = model
    
    def embed_text(self, text: str) -> List[float]:
        """Generate embedding for a single text."""
        response = self.client.embeddings.create(
            model=self.model,
            input=text
        )
        return response.data[0].embedding
    
    def embed_texts(self, texts: List[str]) -> List[List[float]]:
        """Generate embeddings for multiple texts."""
        response = self.client.embeddings.create(
            model=self.model,
            input=texts
        )
        return [item.embedding for item in response.data]
    
    def embed_documents(self, documents: List[Dict]) -> List[Dict]:
        """Generate embeddings for documents."""
        texts = [doc['content'] for doc in documents]
        embeddings = self.embed_texts(texts)
        
        for doc, embedding in zip(documents, embeddings):
            doc['embedding'] = embedding
        
        return documents

28.4.2 Caching Embeddings

Embedding generation can be expensive. Implement caching:

# src/embedding_cache.py
import pickle
import hashlib
from pathlib import Path

class EmbeddingCache:
    def __init__(self, cache_dir="cache/embeddings"):
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(parents=True, exist_ok=True)
    
    def get_cache_key(self, text: str) -> str:
        """Generate a cache key for text."""
        return hashlib.md5(text.encode()).hexdigest()
    
    def get(self, text: str) -> List[float]:
        """Retrieve cached embedding if available."""
        cache_key = self.get_cache_key(text)
        cache_file = self.cache_dir / f"{cache_key}.pkl"
        
        if cache_file.exists():
            with open(cache_file, 'rb') as f:
                return pickle.load(f)
        
        return None
    
    def set(self, text: str, embedding: List[float]):
        """Cache an embedding."""
        cache_key = self.get_cache_key(text)
        cache_file = self.cache_dir / f"{cache_key}.pkl"
        
        with open(cache_file, 'wb') as f:
            pickle.dump(embedding, f)
    
    def embed_with_cache(self, embedding_service, text: str) -> List[float]:
        """Generate embedding with caching."""
        cached = self.get(text)
        if cached is not None:
            return cached
        
        embedding = embedding_service.embed_text(text)
        self.set(text, embedding)
        
        return embedding

# src/vector_store.py
import chromadb
from chromadb.config import Settings
from typing import List, Dict, Optional

class VectorStore:
    def __init__(self, collection_name="documents", persist_directory="./chroma_db"):
        self.client = chromadb.PersistentClient(path=persist_directory)
        self.collection = self.client.get_or_create_collection(
            name=collection_name
        )
    
    def add_documents(self, documents: List[Dict]):
        """Add documents to the vector store."""
        if not documents:
            return
        
        # Prepare data
        ids = [doc['id'] for doc in documents]
        embeddings = [doc['embedding'] for doc in documents]
        texts = [doc['content'] for doc in documents]
        metadatas = [doc.get('metadata', {}) for doc in documents]
        
        # Add to collection
        self.collection.add(
            ids=ids,
            embeddings=embeddings,
            documents=texts,
            metadatas=metadatas
        )
    
    def query(self, query_embedding: List[float], n_results: int = 5, 
              where: Optional[Dict] = None) -> List[Dict]:
        """Query the vector store."""
        results = self.collection.query(
            query_embeddings=[query_embedding],
            n_results=n_results,
            where=where
        )
        
        # Format results
        formatted_results = []
        for i in range(len(results['ids'][0])):
            formatted_results.append({
                'id': results['ids'][0][i],
                'content': results['documents'][0][i],
                'metadata': results['metadatas'][0][i],
                'distance': results['distances'][0][i]
            })
        
        return formatted_results
    
    def delete(self, ids: List[str]):
        """Delete documents from the vector store."""
        self.collection.delete(ids=ids)
    
    def get_collection_info(self) -> Dict:
        """Get information about the collection."""
        return self.collection.count()

# src/pinecone_store.py
from pinecone import Pinecone, ServerlessSpec
import os
from dotenv import load_dotenv

load_dotenv()

class PineconeStore:
    def __init__(self, index_name="documents", dimension=1536):
        self.pc = Pinecone(api_key=os.getenv("PINECONE_API_KEY"))
        self.index_name = index_name
        self.dimension = dimension
        
        # Create index if it doesn't exist
        if index_name not in [index.name for index in self.pc.list_indexes()]:
            self.pc.create_index(
                name=index_name,
                dimension=dimension,
                metric="cosine",
                spec=ServerlessSpec(
                    cloud="aws",
                    region="us-east-1"
                )
            )
        
        self.index = self.pc.Index(index_name)
    
    def upsert_documents(self, documents: List[Dict]):
        """Upsert documents to Pinecone."""
        vectors = []
        
        for doc in documents:
            vectors.append({
                'id': doc['id'],
                'values': doc['embedding'],
                'metadata': {
                    'content': doc['content'],
                    **doc.get('metadata', {})
                }
            })
        
        self.index.upsert(vectors)
    
    def query(self, query_embedding: List[float], top_k: int = 5, 
              filter: Optional[Dict] = None) -> List[Dict]:
        """Query Pinecone index."""
        results = self.index.query(
            vector=query_embedding,
            top_k=top_k,
            filter=filter,
            include_metadata=True
        )
        
        return results['matches']

        )
        
        return results

28.6.2 Hybrid Retrieval

Combine semantic search with keyword search:

# src/hybrid_retriever.py
from src.retriever import Retriever
import re

class HybridRetriever(Retriever):
    def __init__(self, embedding_service: EmbeddingService, 
                 vector_store: VectorStore, bm25_index=None):
        super().__init__(embedding_service, vector_store)
        self.bm25_index = bm25_index
    
    def retrieve_hybrid(self, query: str, n_results: int = 5,
                       semantic_weight: float = 0.7) -> List[Dict]:
        """Retrieve using hybrid semantic and keyword search."""
        # Semantic search
        semantic_results = self.retrieve(query, n_results * 2)
        
        # Keyword search (BM25)
        keyword_results = self.bm25_search(query, n_results * 2) if self.bm25_index else []
        
        # Combine results
        combined = self.combine_results(
            semantic_results, 
            keyword_results, 
            semantic_weight
        )
        
        # Return top results
        return combined[:n_results]
    
    def bm25_search(self, query: str, n_results: int) -> List[Dict]:
        """Perform BM25 keyword search."""
        # Simple keyword matching (in practice, use a proper BM25 library)
        query_terms = set(query.lower().split())
        
        # This is a simplified example
        # In production, use libraries like rank_bm25
        return []
    
    def combine_results(self, semantic_results, keyword_results, 
                       semantic_weight: float) -> List[Dict]:
        """Combine semantic and keyword search results."""
        # Score combination
        combined = {}
        
        # Add semantic scores
        for result in semantic_results:
            doc_id = result['id']
            combined[doc_id] = {
                **result,
                'semantic_score': 1 - result['distance'],  # Convert distance to similarity
                'keyword_score': 0.0
            }
        
        # Add keyword scores
        for result in keyword_results:
            doc_id = result['id']
            if doc_id in combined:
                combined[doc_id]['keyword_score'] = result['score']
            else:
                combined[doc_id] = {
                    **result,
                    'semantic_score': 0.0,
                    'keyword_score': result['score']
                }
        
        # Calculate combined scores
        for doc_id, result in combined.items():
            result['combined_score'] = (
                semantic_weight * result['semantic_score'] +
                (1 - semantic_weight) * result['keyword_score']
            )
        
        # Sort by combined score
        sorted_results = sorted(
            combined.values(),
            key=lambda x: x['combined_score'],
            reverse=True
        )
        
        return sorted_results

28.6.3 Re-ranking

Re-rank retrieved documents for better relevance:

# src/reranker.py
from typing import List

class Reranker:
    def __init__(self, model):
        self.model = model
    
    def rerank(self, query: str, documents: List[Dict], top_k: int = 5) -> List[Dict]:
        """Re-rank documents based on query relevance."""
        if len(documents) <= top_k:
            return documents
        
        # Score each document
        scored_docs = []
        for doc in documents:
            score = self.score_relevance(query, doc['content'])
            scored_docs.append({
                **doc,
                'rerank_score': score
            })
        
        # Sort by rerank score
        reranked = sorted(scored_docs, key=lambda x: x['rerank_score'], reverse=True)
        
        return reranked[:top_k]
    
    def score_relevance(self, query: str, document: str) -> float:
        """Score the relevance of a document to a query."""
        prompt = f"""
On a scale of 0 to 1, rate how relevant this document is to the query.

Query: {query}

Document: {document}

Relevance score (0-1):
"""
        
        response = self.model.generate(prompt)
        
        # Extract score from response
        try:
            score = float(response.strip())
            return min(max(score, 0), 1)
        except:
            return 0.5  # Default score

28.7 Step 5: Context Construction

28.7.1 Context Builder

# src/context_builder.py
from typing import List

class ContextBuilder:
    def __init__(self, max_context_length=3000):
        self.max_context_length = max_context_length
    
    def build_context(self, retrieved_docs: List[Dict]) -> str:
        """Build context from retrieved documents."""
        if not retrieved_docs:
            return ""
        
        # Sort by relevance
        sorted_docs = sorted(
            retrieved_docs,
            key=lambda x: x.get('distance', float('inf'))
        )
        
        # Build context string
        context_parts = []
        current_length = 0
        
        for doc in sorted_docs:
            doc_text = f"[Source: {doc.get('source', 'Unknown')}]\n{doc['content']}\n\n"
            
            if current_length + len(doc_text) > self.max_context_length:
                break
            
            context_parts.append(doc_text)
            current_length += len(doc_text)
        
        return ''.join(context_parts)
    
    def build_context_with_citations(self, retrieved_docs: List[Dict]) -> Dict:
        """Build context with citation information."""
        context = self.build_context(retrieved_docs)
        
        citations = []
        for i, doc in enumerate(retrieved_docs):
            citations.append({
                'id': i + 1,
                'source': doc.get('source', 'Unknown'),
                'metadata': doc.get('metadata', {})
            })
        
        return {
            'context': context,
            'citations': citations
        }

28.8 Step 6: Complete RAG Pipeline

# src/rag_pipeline.py
from src.document_processor import DocumentProcessor
from src.chunker import TextChunker
from src.embedding_service import EmbeddingService
from src.vector_store import VectorStore
from src.retriever import Retriever
from src.context_builder import ContextBuilder
from src.model_client import ModelClient
from src.prompt_loader import load_prompt

class RAGPipeline:
    def __init__(self):
        self.document_processor = DocumentProcessor()
        self.chunker = TextChunker()
        self.embedding_service = EmbeddingService()
        self.vector_store = VectorStore()
        self.retriever = Retriever(self.embedding_service, self.vector_store)
        self.context_builder = ContextBuilder()
        self.model_client = ModelClient()
        
        # Load prompts
        self.system_prompt = load_prompt("rag_system")
        self.answer_template = load_prompt("rag_answer")
    
    def ingest_documents(self, directory: str):
        """Ingest documents into the RAG system."""
        # Process documents
        documents = self.document_processor.ingest_directory(directory)
        
        # Chunk documents
        chunked_docs = []
        for doc in documents:
            chunks = self.chunker.chunk_document(doc)
            chunked_docs.extend(chunks)
        
        # Generate embeddings
        embedded_docs = self.embedding_service.embed_documents(chunked_docs)
        
        # Add to vector store
        self.vector_store.add_documents(embedded_docs)
        
        print(f"Ingested {len(documents)} documents ({len(chunked_docs)} chunks)")
    
    def query(self, user_query: str, n_retrieved: int = 5) -> Dict:
        """Query the RAG system."""
        # Retrieve relevant documents
        retrieved_docs = self.retriever.retrieve(user_query, n_retrieved)
        
        # Build context
        context_data = self.context_builder.build_context_with_citations(retrieved_docs)
        
        # Construct prompt
        user_prompt = self.answer_template.format(
            context=context_data['context'],
            question=user_query
        )
        
        # Generate answer
        answer = self.model_client.generate_with_system(
            self.system_prompt,
            user_prompt,
            temperature=0.3
        )
        
        return {
            'answer': answer,
            'citations': context_data['citations'],
            'retrieved_docs': retrieved_docs
        }

28.9.1 RAG Evaluation Metrics

# src/rag_evaluator.py
from typing import List, Dict

class RAGEvaluator:
    def __init__(self, model_client):
        self.model_client = model_client
    
    def evaluate_retrieval(self, query: str, retrieved_docs: List[Dict],
                          ground_truth_docs: List[str]) -> Dict:
        """Evaluate retrieval quality."""
        retrieved_ids = [doc['id'] for doc in retrieved_docs]
        
        # Precision@k
        precision = len(set(retrieved_ids) & set(ground_truth_docs)) / len(retrieved_ids)
        
        # Recall@k
        recall = len(set(retrieved_ids) & set(ground_truth_docs)) / len(ground_truth_docs)
        
        # MRR (Mean Reciprocal Rank)
        mrr = 0.0
        for i, doc_id in enumerate(retrieved_ids):
            if doc_id in ground_truth_docs:
                mrr = 1.0 / (i + 1)
                break
        
        return {
            'precision': precision,
            'recall': recall,
            'mrr': mrr
        }
    
    def evaluate_answer(self, query: str, answer: str, 
                       ground_truth: str) -> Dict:
        """Evaluate answer quality."""
        prompt = f"""
Evaluate this answer on a scale of 0 to 1 for:
1. Factual accuracy
2. Completeness
3. Clarity

Question: {query}

Answer: {answer}

Ground truth: {ground_truth}

Provide scores in JSON format:
{{"accuracy": 0.0-1.0, "completeness": 0.0-1.0, "clarity": 0.0-1.0}}
"""
        
        response = self.model_client.generate(prompt)
        
        try:
            import json
            scores = json.loads(response)
            return scores
        except:
            return {
                'accuracy': 0.5,
                'completeness': 0.5,
                'clarity': 0.5
            }
    
    def evaluate_rag(self, query: str, retrieved_docs: List[Dict],
                    answer: str, ground_truth_docs: List[str],
                    ground_truth_answer: str) -> Dict:
        """Evaluate complete RAG pipeline."""
        retrieval_metrics = self.evaluate_retrieval(query, retrieved_docs, ground_truth_docs)
        answer_metrics = self.evaluate_answer(query, answer, ground_truth_answer)
        
        return {
            'retrieval': retrieval_metrics,
            'answer': answer_metrics
        }

def optimize_chunk_size(documents, test_queries, size_range=[200, 300, 400, 500, 600]):

    results = {}

        chunked_docs = []

        avg_precision = evaluate_retrieval(chunked_docs, test_queries)

    # Return best chunk size
    return max(results.items(), key=lambda x: x[1])

28.11 Best Practices

28.11.1 1. Document Quality

  • Clean and preprocess documents before ingestion
  • Remove duplicate content
  • Normalize text formatting
  • Extract meaningful metadata

28.11.2 2. Chunking Strategy

  • Choose chunk size based on use case
  • Use overlap to maintain context
  • Consider semantic chunking for complex documents
  • Test different strategies

28.11.3 3. Retrieval Optimization

  • Use hybrid retrieval for better results
  • Implement re-ranking for precision
  • Filter by metadata when appropriate
  • Cache frequently accessed results

28.11.4 4. Evaluation

  • Establish ground truth datasets
  • Measure both retrieval and answer quality
  • Track performance over time
  • Use A/B testing for improvements

28.11.5 5. Production Considerations

  • Monitor retrieval latency
  • Track embedding costs
  • Implement caching strategies
  • Plan for scale

28.12 Common Pitfalls

28.12.1 1. Poor Chunking

Problem: Chunks are too large or too small, reducing retrieval quality.

Solution: Test different chunk sizes and overlaps. Use semantic chunking for complex documents.

28.12.2 2. Insufficient Context

Problem: Retrieved context doesn’t contain enough information to answer the query.

Solution: Increase retrieval count, improve chunking, or use re-ranking.

28.12.3 3. Context Overflow

Problem: Too much context causes the model to lose focus or exceed token limits.

Solution: Implement context length limits, prioritize relevant chunks, use re-ranking.

28.12.4 4. Ignoring Metadata

Problem: Not using metadata filters leads to irrelevant results.

Solution: Extract and use metadata for filtering (date, category, source).

28.12.5 5. No Evaluation

Problem: Without evaluation, you can’t measure improvements.

Solution: Build evaluation from the start, track metrics continuously.

28.13 Conclusion

Building effective RAG systems requires integrating multiple components: document processing, embedding generation, vector storage, retrieval, and answer generation. Each component must be carefully designed and optimized to work together effectively.

The key to success is iterative improvement: start with a basic pipeline, measure performance, identify bottlenecks, and optimize systematically. Use evaluation metrics to guide decisions and track progress over time.
