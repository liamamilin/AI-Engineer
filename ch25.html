30  Chapter 25 — Evaluation and Debugging AI Systems

30.1 Introduction

Chapter 12 established evaluation as a fundamental discipline in AI engineering—measuring system behavior across distributions of inputs and conditions rather than verifying deterministic correctness. However, knowing that evaluation is important is different from knowing how to implement it effectively.

This chapter provides practical techniques for evaluating and debugging AI systems. It covers building evaluation frameworks, implementing automated and human evaluation, debugging probabilistic systems, and using tools and observability platforms to gain insight into system behavior.

30.2 Building Evaluation Frameworks

30.2.1 Framework Architecture

A robust evaluation framework provides structure, repeatability, and visibility:

evaluation framework
├─ dataset management
├─ metric calculation
├─ result analysis
└─ reporting

30.2.2 Core Components

# src/evaluation/framework.py
from typing import List, Dict, Callable, Any
from datetime import datetime
import json

class EvaluationFramework:
    def __init__(self):
        self.datasets = {}
        self.metrics = {}
        self.results = []
    
    def register_dataset(self, name: str, data: List[Dict]):
        """Register an evaluation dataset."""
        self.datasets[name] = {
            'data': data,
            'created_at': datetime.now()
        }
    
    def register_metric(self, name: str, metric_fn: Callable):
        """Register an evaluation metric."""
        self.metrics[name] = metric_fn
    
    def evaluate(self, system: Callable, dataset_name: str, 
                 metrics: List[str]) -> Dict:
        """Evaluate a system on a dataset."""
        if dataset_name not in self.datasets:
            raise ValueError(f"Dataset {dataset_name} not found")
        
        dataset = self.datasets[dataset_name]['data']
        results = []
        
        for example in dataset:
            # Get system output
            output = system(example['input'])
            
            # Calculate metrics
            metric_results = {}
            for metric_name in metrics:
                if metric_name in self.metrics:
                    metric_fn = self.metrics[metric_name]
                    metric_value = metric_fn(
                        example['input'],
                        output,
                        example.get('expected', None)
                    )
                    metric_results[metric_name] = metric_value
            
            results.append({
                'input': example['input'],
                'output': output,
                'expected': example.get('expected'),
                'metrics': metric_results,
                'timestamp': datetime.now()
            })
        
        # Aggregate results
        aggregated = self.aggregate_results(results, metrics)
        
        # Store results
        evaluation_result = {
            'dataset': dataset_name,
            'metrics': metrics,
            'results': results,
            'aggregated': aggregated,
            'timestamp': datetime.now()
        }
        
        self.results.append(evaluation_result)
        
        return evaluation_result
    
    def aggregate_results(self, results: List[Dict], 
                         metrics: List[str]) -> Dict:
        """Aggregate evaluation results."""
        aggregated = {}
        
        for metric_name in metrics:
            values = [r['metrics'][metric_name] for r in results]
            
            aggregated[metric_name] = {
                'mean': sum(values) / len(values),
                'min': min(values),
                'max': max(values),
                'std': self.calculate_std(values)
            }
        
        return aggregated
    
    def calculate_std(self, values: List[float]) -> float:
        """Calculate standard deviation."""
        import statistics
        return statistics.stdev(values) if len(values) > 1 else 0.0
    
    def save_results(self, results: Dict, filepath: str):
        """Save evaluation results to file."""
        with open(filepath, 'w') as f:
            json.dump(results, f, default=str, indent=2)

30.2.3 Dataset Management

# src/evaluation/dataset.py
from typing import List, Dict
import json

class Dataset:
    def __init__(self, name: str, description: str = ""):
        self.name = name
        self.description = description
        self.examples = []
        self.metadata = {}
    
    def add_example(self, input_data: Any, expected: Any = None,
                   metadata: Dict = None):
        """Add an example to the dataset."""
        self.examples.append({
            'input': input_data,
            'expected': expected,
            'metadata': metadata or {}
        })
    
    def load_from_file(self, filepath: str):
        """Load dataset from JSON file."""
        with open(filepath, 'r') as f:
            data = json.load(f)
        
        self.name = data.get('name', self.name)
        self.description = data.get('description', '')
        self.metadata = data.get('metadata', {})
        self.examples = data.get('examples', [])
    
    def save_to_file(self, filepath: str):
        """Save dataset to JSON file."""
        data = {
            'name': self.name,
            'description': self.description,
            'metadata': self.metadata,
            'examples': self.examples
        }
        
        with open(filepath, 'w') as f:
            json.dump(data, f, indent=2)
    
    def split(self, train_ratio: float = 0.7, 
              val_ratio: float = 0.15) -> Dict[str, 'Dataset']:
        """Split dataset into train, validation, and test sets."""
        import random
        
        shuffled = self.examples.copy()
        random.shuffle(shuffled)
        
        n = len(shuffled)
        train_end = int(n * train_ratio)
        val_end = train_end + int(n * val_ratio)
        
        train_data = Dataset(f"{self.name}_train")
        val_data = Dataset(f"{self.name}_val")
        test_data = Dataset(f"{self.name}_test")
        
        train_data.examples = shuffled[:train_end]
        val_data.examples = shuffled[train_end:val_end]
        test_data.examples = shuffled[val_end:]
        
        return {
            'train': train_data,
            'val': val_data,
            'test': test_data
        }

30.3.1 Text Similarity Metrics

# src/evaluation/metrics/text.py

def exact_match(input_data, output, expected):
    """Exact match metric."""
    return 1.0 if str(output).strip() == str(expected).strip() else 0.0

def keyword_overlap(input_data, output, expected):
    """Keyword overlap metric."""
    output_words = set(str(output).lower().split())
    expected_words = set(str(expected).lower().split())
    
    if not expected_words:
        return 1.0 if not output_words else 0.0
    
    overlap = output_words & expected_words
    return len(overlap) / len(expected_words)

def bleu_score(input_data, output, expected):
    """BLEU score for text generation."""
    try:
        from sacrebleu import corpus_bleu
        return corpus_bleu([str(output)], [[str(expected)]]).score / 100.0
    except:
        return 0.0

def rouge_score(input_data, output, expected):
    """ROUGE score for summarization."""
    try:
        from rouge import Rouge
        rouge = Rouge()
        scores = rouge.get_scores(str(output), str(expected))
        return scores[0]['rouge-l']['f']
    except:
        return 0.0

def semantic_similarity(input_data, output, expected):
    """Semantic similarity using embeddings."""
    try:
        from sentence_transformers import SentenceTransformer
        model = SentenceTransformer('all-MiniLM-L6-v2')
        
        emb1 = model.encode(str(output))
        emb2 = model.encode(str(expected))
        
        from sklearn.metrics.pairwise import cosine_similarity
        return cosine_similarity([emb1], [emb2])[0][0]
    except:
        return 0.0

def qa_f1(input_data, output, expected):
    """F1 score for question answering."""
    import string
    from collections import Counter
    
    def normalize_answer(s):
        """Lower text and remove punctuation, articles and extra whitespace."""
        def remove_articles(text):
            return re.sub(r'\b(a|an|the)\b', ' ', text)
        
        def white_space_fix(text):
            return ' '.join(text.split())
        
        def remove_punc(text):
            exclude = set(string.punctuation)
            return ''.join(ch for ch in text if ch not in exclude)
        
        def lower(text):
            return text.lower()
        
        return white_space_fix(remove_articles(remove_punc(lower(s))))
    
    prediction_tokens = normalize_answer(str(output)).split()
    reference_tokens = normalize_answer(str(expected)).split()
    
    common = Counter(prediction_tokens) & Counter(reference_tokens)
    num_same = sum(common.values())
    
    if len(reference_tokens) == 0 or len(prediction_tokens) == 0:
        return int(reference_tokens == prediction_tokens)
    
    precision = 1.0 * num_same / len(prediction_tokens)
    recall = 1.0 * num_same / len(reference_tokens)
    f1 = (2 * precision * recall) / (precision + recall)
    
    return f1

def classification_accuracy(input_data, output, expected):
    """Accuracy for classification tasks."""
    return 1.0 if str(output).lower() == str(expected).lower() else 0.0

def information_retrieval_metrics(input_data, output, expected):
    """Metrics for information retrieval (precision, recall, F1)."""
    retrieved = set(output) if isinstance(output, list) else {output}
    relevant = set(expected) if isinstance(expected, list) else {expected}
    
    if not relevant:
        return {'precision': 1.0, 'recall': 1.0, 'f1': 1.0}
    
    true_positives = len(retrieved & relevant)
    
    precision = true_positives / len(retrieved) if retrieved else 0.0
    recall = true_positives / len(relevant)
    f1 = 2 * (precision * recall) / (precision + recall) if (precision + recall) > 0 else 0.0
    
    return {
        'precision': precision,
        'recall': recall,
        'f1': f1
    }

def code_generation_accuracy(input_data, output, expected):
    """Accuracy for code generation tasks."""
    try:
        # Try to execute both outputs
        exec_globals = {}
        exec(str(output), exec_globals)
        exec(str(expected), exec_globals)
        
        # Compare execution results
        return 1.0  # If both execute without error
    except:
        return 0.0

30.4.1 Continuous Evaluation Pipeline

# src/evaluation/continuous.py

class ContinuousEvaluator:
    def __init__(self, framework, system):
        self.framework = framework
        self.system = system
        self.history = []
    
    def run_evaluation(self, dataset_name: str, metrics: List[str]):
        """Run evaluation and store results."""
        results = self.framework.evaluate(
            self.system,
            dataset_name,
            metrics
        )
        
        self.history.append({
            'timestamp': datetime.now(),
            'results': results
        })
        
        return results
    
    def compare_runs(self, run1_idx: int, run2_idx: int):
        """Compare two evaluation runs."""
        run1 = self.history[run1_idx]['results']
        run2 = self.history[run2_idx]['results']
        
        comparison = {}
        
        for metric in run1['aggregated']:
            comparison[metric] = {
                'run1': run1['aggregated'][metric]['mean'],
                'run2': run2['aggregated'][metric]['mean'],
                'difference': run2['aggregated'][metric]['mean'] - run1['aggregated'][metric]['mean'],
                'percent_change': (
                    (run2['aggregated'][metric]['mean'] - run1['aggregated'][metric]['mean']) /
                    run1['aggregated'][metric]['mean'] * 100
                    if run1['aggregated'][metric]['mean'] != 0 else 0
                )
            }
        
        return comparison
    
    def detect_regression(self, threshold: float = 0.05):
        """Detect performance regression compared to previous run."""
        if len(self.history) < 2:
            return None
        
        current_idx = len(self.history) - 1
        previous_idx = len(self.history) - 2
        
        comparison = self.compare_runs(previous_idx, current_idx)
        
        regressions = []
        for metric, data in comparison.items():
            if data['percent_change'] < -threshold:
                regressions.append({
                    'metric': metric,
                    'previous': data['run1'],
                    'current': data['run2'],
                    'change': data['percent_change']
                })
        
        return regressions if regressions else None

30.4.2 A/B Testing

# src/evaluation/ab_testing.py

class ABTester:
    def __init__(self, framework):
        self.framework = framework
    
    def test(self, system_a, system_b, dataset_name: str, 
            metrics: List[str]) -> Dict:
        """Run A/B test between two systems."""
        # Evaluate system A
        results_a = self.framework.evaluate(
            system_a,
            dataset_name,
            metrics
        )
        
        # Evaluate system B
        results_b = self.framework.evaluate(
            system_b,
            dataset_name,
            metrics
        )
        
        # Compare results
        comparison = self.compare_results(results_a, results_b, metrics)
        
        return {
            'system_a': results_a,
            'system_b': results_b,
            'comparison': comparison
        }
    
    def compare_results(self, results_a, results_b, metrics: List[str]) -> Dict:
        """Compare evaluation results."""
        comparison = {}
        
        for metric in metrics:
            mean_a = results_a['aggregated'][metric]['mean']
            mean_b = results_b['aggregated'][metric]['mean']
            
            comparison[metric] = {
                'system_a': mean_a,
                'system_b': mean_b,
                'difference': mean_b - mean_a,
                'winner': 'A' if mean_a > mean_b else 'B' if mean_b > mean_a else 'tie'
            }
        
        return comparison

30.6.3 Tool and Integration Debugging

# src/debugging/tool_debugger.py

class ToolDebugger:
    def __init__(self):
        self.tool_calls = []
    
    def instrument_tool(self, tool: Callable) -> Callable:
        """Instrument a tool for debugging."""
        def wrapped_tool(*args, **kwargs):
            start_time = datetime.now()
            
            try:
                result = tool(*args, **kwargs)
                
                self.tool_calls.append({
                    'tool': tool.__name__,
                    'args': args,
                    'kwargs': kwargs,
                    'result': result,
                    'error': None,
                    'duration': (datetime.now() - start_time).total_seconds(),
                    'success': True
                })
                
                return result
            
            except Exception as e:
                self.tool_calls.append({
                    'tool': tool.__name__,
                    'args': args,
                    'kwargs': kwargs,
                    'result': None,
                    'error': str(e),
                    'duration': (datetime.now() - start_time).total_seconds(),
                    'success': False
                })
                
                raise
        
        return wrapped_tool
    
    def get_tool_statistics(self) -> Dict:
        """Get statistics about tool usage."""
        if not self.tool_calls:
            return {}
        
        stats = {}
        
        for call in self.tool_calls:
            tool_name = call['tool']
            
            if tool_name not in stats:
                stats[tool_name] = {
                    'calls': 0,
                    'successes': 0,
                    'failures': 0,
                    'total_duration': 0
                }
            
            stats[tool_name]['calls'] += 1
            if call['success']:
                stats[tool_name]['successes'] += 1
            else:
                stats[tool_name]['failures'] += 1
            stats[tool_name]['total_duration'] += call['duration']
        
        return stats

            run_id=run_id

class WandBIntegration:

                    type: str = "dataset"):
        """Log an artifact to W&B."""
        artifact = self.run.Artifact(name, type=type)
        artifact.add_file(artifact_path)
        self.run.log_artifact(artifact)
    
    def log_table(self, table_name: str, data: List[Dict]):
        """Log a table to W&B."""
        table = self.run.Table(columns=list(data[0].keys()))

            table.add_data(*row.values())
        self.run.log({table_name: table})

30.8 Best Practices

30.8.1 1. Define Clear Success Criteria

Before evaluating, define what success looks like for your system. This guides metric selection and interpretation.

30.8.2 2. Use Multiple Metrics

No single metric captures all aspects of system performance. Use a combination of metrics to get a complete picture.

30.8.3 3. Maintain Evaluation Datasets

Keep versioned evaluation datasets that represent your use cases. Update them as requirements evolve.

30.8.4 4. Automate Evaluation

Integrate evaluation into your CI/CD pipeline to catch regressions early.

30.8.5 5. Combine Automated and Human Evaluation

Automated metrics are efficient, but human evaluation catches issues that automated metrics miss.

30.8.6 6. Monitor Continuously

Don’t evaluate only during development. Monitor performance in production to detect drift.

30.9 Common Pitfalls

30.9.1 1. Overfitting to Evaluation Metrics

Problem: Optimizing for metrics at the expense of actual user value.

Solution: Use human evaluation to validate that metric improvements translate to real value.

30.9.2 2. Inadequate Test Coverage

Problem: Evaluation datasets don’t cover important edge cases.

Solution: Continuously expand evaluation datasets with real-world examples.

30.9.3 3. Ignoring Context

Problem: Evaluating outputs without considering the context in which they were generated.

Solution: Log and analyze the full execution context, not just inputs and outputs.

30.9.4 4. Metric Gaming

Problem: Systems learn to exploit specific metrics rather than genuinely improving.

Solution: Use diverse metrics and regularly validate against human judgment.

30.9.5 5. Lack of Baseline

Problem: No baseline to compare against, making it hard to assess improvement.

Solution: Always maintain baseline systems for comparison.

30.10 Conclusion

Effective evaluation and debugging are essential for building reliable AI systems. By implementing robust evaluation frameworks, using a combination of automated and human evaluation, and maintaining comprehensive observability, engineers can systematically improve system performance and catch issues early.

The techniques in this chapter—framework design, metric implementation, continuous evaluation, and debugging tools—provide a practical toolkit for ensuring AI systems behave as intended and can be improved over time.
