29  Chapter 24 — Designing Multi-Agent Systems

29.1 Introduction

Chapter 10 introduced agents as computational entities capable of observation, reasoning, and action through iterative loops. Single agents excel at autonomous problem-solving, but complex tasks often require coordination among multiple specialized agents.

Multi-agent systems extend the agent paradigm by enabling collaboration, competition, and communication among autonomous entities. Just as teams of humans with different expertise can solve complex problems more effectively than individuals, multi-agent AI systems can tackle tasks that exceed the capabilities of any single agent.

This chapter explores the design and implementation of multi-agent systems, covering communication patterns, coordination mechanisms, architectural patterns, and practical implementation strategies.

29.2 The Multi-Agent Paradigm

29.2.1 Why Multiple Agents?

Multi-agent systems offer several advantages over single-agent approaches:

Specialization: Each agent can focus on a specific domain or task, developing deep expertise in its area of responsibility.

Parallelization: Agents can work simultaneously on different aspects of a problem, reducing overall completion time.

Robustness: If one agent fails, others can continue functioning, making the system more resilient.

Scalability: New agents can be added to handle increased complexity or new requirements.

Natural Modeling: Many real-world problems are inherently multi-agent (teams, organizations, markets).

29.2.2 Agent Types

Multi-agent systems typically include different types of agents:

Specialist Agents: Focused on specific tasks (researcher, writer, analyst, critic)

Coordinator Agents: Orchestrate interactions among specialists

Manager Agents: Oversee high-level strategy and resource allocation

Monitor Agents: Observe system behavior and report issues

Utility Agents: Provide common services (logging, evaluation, persistence)

29.3 Communication Patterns

29.3.1 Pattern 1: Hierarchical Communication

Agents communicate through a defined hierarchy:

Manager Agent
    ↓ (assigns tasks)
Specialist Agents
    ↓ (report results)
Manager Agent
    ↓ (synthesizes)
Final Output

Implementation:

# src/agents/manager.py
from src.agents.base_agent import BaseAgent

class ManagerAgent(BaseAgent):
    def __init__(self, specialists, model_client):
        super().__init__(model_client)
        self.specialists = specialists
    
    def orchestrate(self, task):
        """Orchestrate specialists to complete a task."""
        # Analyze task
        analysis = self.analyze_task(task)
        
        # Decompose into subtasks
        subtasks = self.decompose_task(analysis)
        
        # Assign to specialists
        results = {}
        for subtask in subtasks:
            specialist = self.select_specialist(subtask)
            result = specialist.execute(subtask)
            results[subtask['id']] = result
        
        # Synthesize results
        final_output = self.synthesize_results(results)
        
        return final_output
    
    def analyze_task(self, task):
        """Analyze the task requirements."""
        prompt = f"""
Analyze this task and identify what needs to be done:
{task}

Provide:
1. Task complexity
2. Required capabilities
3. Subtasks needed
4. Dependencies
"""
        return self.think(prompt)
    
    def decompose_task(self, analysis):
        """Decompose task into subtasks."""
        prompt = f"""
Based on this analysis, decompose the task into subtasks:
{analysis}

For each subtask, provide:
- ID
- Description
- Required capabilities
- Dependencies
"""
        return self.think(prompt)
    
    def select_specialist(self, subtask):
        """Select appropriate specialist for subtask."""
        capabilities = subtask['required_capabilities']
        
        for specialist in self.specialists:
            if specialist.can_handle(capabilities):
                return specialist
        
        raise Exception(f"No specialist available for: {capabilities}")
    
    def synthesize_results(self, results):
        """Synthesize results from specialists."""
        prompt = f"""
Synthesize these results into a coherent output:
{results}

Provide a comprehensive, well-structured response.
"""
        return self.think(prompt)

# src/agents/peer_agent.py
from src.agents.base_agent import BaseAgent
from src.communication.message_bus import MessageBus

class PeerAgent(BaseAgent):
    def __init__(self, agent_id, capabilities, model_client, message_bus):
        super().__init__(model_client)
        self.agent_id = agent_id
        self.capabilities = capabilities
        self.message_bus = message_bus
        self.message_bus.subscribe(self.agent_id, self.handle_message)
    
    def handle_message(self, message):
        """Handle incoming message from another agent."""
        print(f"{self.agent_id} received: {message}")
        
        # Process message
        response = self.process_message(message)
        
        # Send response if needed
        if response:
            self.message_bus.send(
                from_agent=self.agent_id,
                to_agent=message['from'],
                content=response
            )
    
    def process_message(self, message):
        """Process message and generate response."""
        prompt = f"""
You are {self.agent_id}, an agent with capabilities: {self.capabilities}

You received this message from {message['from']}:
{message['content']}

Respond appropriately based on your capabilities.
"""
        return self.think(prompt)
    
    def broadcast(self, content):
        """Broadcast message to all agents."""
        self.message_bus.broadcast(
            from_agent=self.agent_id,
            content=content
        )
    
    def send_to(self, to_agent, content):
        """Send message to specific agent."""
        self.message_bus.send(
            from_agent=self.agent_id,
            to_agent=to_agent,
            content=content
        )

Message Bus Implementation:

# src/communication/message_bus.py
from typing import Dict, Callable

class MessageBus:
    def __init__(self):
        self.subscribers: Dict[str, Callable] = {}
        self.message_log = []
    
    def subscribe(self, agent_id: str, handler: Callable):
        """Subscribe an agent to receive messages."""
        self.subscribers[agent_id] = handler
    
    def send(self, from_agent: str, to_agent: str, content: str):
        """Send message to specific agent."""
        message = {
            'from': from_agent,
            'to': to_agent,
            'content': content,
            'timestamp': datetime.now()
        }
        
        self.message_log.append(message)
        
        if to_agent in self.subscribers:
            self.subscribers[to_agent](message)
    
    def broadcast(self, from_agent: str, content: str):
        """Broadcast message to all agents."""
        for agent_id in self.subscribers:
            if agent_id != from_agent:
                self.send(from_agent, agent_id, content)

29.3.3 Pattern 3: Blackboard Architecture

Agents share information through a common workspace:

Agents
    ↓ (read/write)
Blackboard (Shared Knowledge)
    ↑ (read/write)
Agents

Implementation:

# src/communication/blackboard.py
from typing import Dict, List, Any

class Blackboard:
    def __init__(self):
        self.data: Dict[str, Any] = {}
        self.history: List[Dict] = []
    
    def write(self, key: str, value: Any, agent_id: str):
        """Write data to blackboard."""
        self.data[key] = {
            'value': value,
            'agent_id': agent_id,
            'timestamp': datetime.now()
        }
        
        self.history.append({
            'action': 'write',
            'key': key,
            'agent_id': agent_id,
            'timestamp': datetime.now()
        })
    
    def read(self, key: str) -> Any:
        """Read data from blackboard."""
        if key in self.data:
            return self.data[key]['value']
        return None
    
    def read_all(self) -> Dict[str, Any]:
        """Read all data from blackboard."""
        return {k: v['value'] for k, v in self.data.items()}
    
    def get_history(self) -> List[Dict]:
        """Get blackboard history."""
        return self.history

Blackboard Agent:

# src/agents/blackboard_agent.py
from src.agents.base_agent import BaseAgent
from src.communication.blackboard import Blackboard

class BlackboardAgent(BaseAgent):
    def __init__(self, agent_id, blackboard, model_client):
        super().__init__(model_client)
        self.agent_id = agent_id
        self.blackboard = blackboard
    
    def execute(self, task):
        """Execute task using blackboard."""
        # Read relevant data from blackboard
        context = self.read_context(task)
        
        # Process task
        result = self.process_task(task, context)
        
        # Write result to blackboard
        self.write_result(task, result)
        
        return result
    
    def read_context(self, task):
        """Read relevant context from blackboard."""
        prompt = f"""
You are {self.agent_id}. You need to complete this task:
{task}

Based on the blackboard data:
{self.blackboard.read_all()}

What information do you need from the blackboard?
"""
        needs = self.think(prompt)
        
        # Read relevant keys
        context = {}
        for key in self.parse_keys(needs):
            context[key] = self.blackboard.read(key)
        
        return context
    
    def process_task(self, task, context):
        """Process task with context."""
        prompt = f"""
Complete this task:
{task}

Using this context:
{context}
"""
        return self.think(prompt)
    
    def write_result(self, task, result):
        """Write result to blackboard."""
        key = f"{task}_{datetime.now().strftime('%Y%m%d_%H%M%S')}"
        self.blackboard.write(key, result, self.agent_id)

29.4 Coordination Mechanisms

29.4.1 Mechanism 1: Task Allocation

Distribute tasks among agents based on capabilities and load:

# src/coordination/task_allocator.py
from typing import List, Dict

class TaskAllocator:
    def __init__(self, agents):
        self.agents = agents
        self.task_queue = []
    
    def allocate_task(self, task):
        """Allocate task to most suitable agent."""
        # Score each agent
        scored_agents = []
        for agent in self.agents:
            score = self.score_agent(agent, task)
            scored_agents.append((agent, score))
        
        # Select best agent
        best_agent = max(scored_agents, key=lambda x: x[1])[0]
        
        # Assign task
        result = best_agent.execute(task)
        
        return result
    
    def score_agent(self, agent, task):
        """Score agent's suitability for task."""
        score = 0
        
        # Capability match
        if agent.can_handle(task['required_capabilities']):
            score += 50
        
        # Current load (lower is better)
        score -= agent.current_load * 10
        
        # Historical performance
        score += agent.performance_score * 20
        
        return score

29.4.2 Mechanism 2: Consensus Building

Agents reach agreement through discussion:

# src/coordination/consensus.py
from src.agents.base_agent import BaseAgent

class ConsensusBuilder:
    def __init__(self, agents, model_client):
        self.agents = agents
        self.model_client = model_client
    
    def build_consensus(self, proposal):
        """Build consensus among agents on a proposal."""
        # Get initial opinions
        opinions = {}
        for agent in self.agents:
            opinion = agent.evaluate(proposal)
            opinions[agent.agent_id] = opinion
        
        # Facilitate discussion
        rounds = 0
        max_rounds = 3
        
        while rounds < max_rounds:
            # Share opinions
            for agent in self.agents:
                agent.receive_opinions(opinions)
            
            # Get updated opinions
            new_opinions = {}
            for agent in self.agents:
                opinion = agent.evaluate(proposal)
                new_opinions[agent.agent_id] = opinion
            
            # Check for convergence
            if self.check_convergence(opinions, new_opinions):
                break
            
            opinions = new_opinions
            rounds += 1
        
        # Synthesize consensus
        consensus = self.synthesize_consensus(opinions)
        
        return consensus
    
    def check_convergence(self, old_opinions, new_opinions):
        """Check if opinions have converged."""
        # Simple check: are opinions similar?
        for agent_id in old_opinions:
            if not self.similar_opinions(
                old_opinions[agent_id],
                new_opinions[agent_id]
            ):
                return False
        return True
    
    def similar_opinions(self, opinion1, opinion2):
        """Check if two opinions are similar."""
        # Use model to evaluate similarity
        prompt = f"""
Are these two opinions similar?

Opinion 1: {opinion1}
Opinion 2: {opinion2}

Respond with "yes" or "no".
"""
        response = self.model_client.generate(prompt)
        return "yes" in response.lower()
    
    def synthesize_consensus(self, opinions):
        """Synthesize consensus from opinions."""
        prompt = f"""
Synthesize a consensus from these opinions:
{opinions}

Provide a unified position that incorporates the perspectives of all agents.
"""
        return self.model_client.generate(prompt)

29.5.2 Pattern 2: Competitive Debate

Multiple agents debate to reach best solution:

# src/patterns/competitive_debate.py

class DebateSystem:
    def __init__(self, debaters, judge, model_client):
        self.debaters = debaters
        self.judge = judge
        self.model_client = model_client
    
    def conduct_debate(self, topic):
        """Conduct a debate on a topic."""
        # Initial arguments
        arguments = {}
        for debater in self.debaters:
            argument = debater.make_argument(topic)
            arguments[debater.agent_id] = argument
        
        # Counter-arguments
        rounds = 0
        max_rounds = 2
        
        while rounds < max_rounds:
            for debater in self.debaters:
                counter_argument = debater.make_counter_argument(
                    topic,
                    arguments
                )
                arguments[debater.agent_id] = counter_argument
            
            rounds += 1
        
        # Judge evaluates
        verdict = self.judge.evaluate_debate(topic, arguments)
        
        return {
            'arguments': arguments,
            'verdict': verdict
        }

29.5.3 Pattern 3: Swarm Intelligence

Decentralized agents solve problems through emergent behavior:

# src/patterns/swarm.py

class SwarmAgent(BaseAgent):
    def __init__(self, agent_id, neighbors, model_client):
        super().__init__(model_client)
        self.agent_id = agent_id
        self.neighbors = neighbors
        self.local_state = {}
    
    def sense(self):
        """Sense local environment."""
        # Gather information from neighbors
        local_info = {}
        for neighbor in self.neighbors:
            info = neighbor.share_state()
            local_info[neighbor.agent_id] = info
        
        return local_info
    
    def decide(self, local_info):
        """Decide on action based on local information."""
        prompt = f"""
You are {self.agent_id} in a swarm.

Your local state: {self.local_state}

Information from neighbors: {local_info}

Decide on your next action.
"""
        return self.think(prompt)
    
    def act(self, action):
        """Execute action and update state."""
        self.local_state['last_action'] = action
        # ... (update state)
    
    def share_state(self):
        """Share state with neighbors."""
        return self.local_state
    
    def run(self):
        """Run swarm behavior loop."""
        while True:
            # Sense
            local_info = self.sense()
            
            # Decide
            action = self.decide(local_info)
            
            # Act
            self.act(action)
            
            # Check termination
            if self.should_terminate():
                break

class ResearcherAgent(BaseAgent):
    def __init__(self, model_client, tools):
        super().__init__(model_client)
        self.tools = tools
    
    def execute(self, task):
        """Execute research task."""
        # Understand research question
        research_plan = self.plan_research(task['question'])
        
        # Conduct research
        findings = []
        for step in research_plan:
            result = self.execute_research_step(step)
            findings.append(result)
        
        # Synthesize findings
        synthesis = self.synthesize_findings(findings)
        
        return synthesis
    
    def plan_research(self, question):

Plan research for this question:
{question}

Available tools: {self.tool_names}

Create a step-by-step research plan.
"""
        return self.think(prompt)
    
    def execute_research_step(self, step):
        """Execute a single research step."""
        tool_name = step['tool']
        tool = self.tools[tool_name]
        
        return tool.execute(step['parameters'])
    
    def synthesize_findings(self, findings):
        """Synthesize research findings."""
        prompt = f"""
Synthesize these research findings:
{findings}

Provide a comprehensive summary.
"""
        return self.think(prompt)

        self.critic = CriticAgent(model_client)
    
    def research_topic(self, question):
        """Research a topic using multi-agent collaboration."""
        print(f"Researching: {question}")
        
        # Phase 1: Research
        print("Phase 1: Conducting research...")
        research_task = {'question': question}
        findings = self.researcher.execute(research_task)
        
        # Phase 2: Analysis
        print("Phase 2: Analyzing findings...")
        analysis_task = {
            'findings': findings
        }
        analysis = self.analyst.execute(analysis_task)
        
        # Phase 3: Writing
        print("Phase 3: Writing report...")
        writing_task = {
            'topic': question,
            'analysis': analysis
        }
        report = self.writer.execute(writing_task)
        
        # Phase 4: Critique
        print("Phase 4: Critiquing report...")
        critique_task = {'report': report}
        critique = self.critic.execute(critique_task)
        
        # Phase 5: Revision (if needed)
        if critique['rating'] < 7:
            print("Phase 5: Revising report...")
            report = self.revise_report(report, critique)
        
        return {
            'findings': findings,
            'analysis': analysis,
            'report': report,
            'critique': critique
        }
    
    def revise_report(self, report, critique):
        """Revise report based on critique."""
        prompt = f"""
Revise this report based on the critique:
{report}

Critique:
{critique}

Provide the revised report.
"""
        return self.model_client.generate(prompt)

29.7.1 Observation and Logging

class AgentObserver:
    def __init__(self):
        self.logs = []
    
    def log_action(self, agent_id, action, result):
        """Log an agent action."""
        self.logs.append({
            'agent_id': agent_id,
            'action': action,
            'result': result,
            'timestamp': datetime.now()
        })
    
    def get_agent_logs(self, agent_id):
        """Get logs for a specific agent."""
        return [log for log in self.logs if log['agent_id'] == agent_id]
    
    def get_all_logs(self):
        """Get all logs."""
        return self.logs
    
    def visualize_execution(self):
        """Visualize execution flow."""
        # Create a timeline of agent interactions
        timeline = []
        for log in self.logs:
            timeline.append({
                'time': log['timestamp'],
                'agent': log['agent_id'],
                'action': log['action']
            })
        
        return timeline

29.7.2 Performance Monitoring

# src/debugging/performance_monitor.py

class PerformanceMonitor:
    def __init__(self):
        self.metrics = {}
    
    def record_metric(self, agent_id, metric_name, value):
        """Record a performance metric."""
        if agent_id not in self.metrics:
            self.metrics[agent_id] = {}
        
        if metric_name not in self.metrics[agent_id]:
            self.metrics[agent_id][metric_name] = []
        
        self.metrics[agent_id][metric_name].append(value)
    
    def get_agent_metrics(self, agent_id):
        """Get metrics for a specific agent."""
        return self.metrics.get(agent_id, {})
    
    def get_average_metric(self, agent_id, metric_name):
        """Get average value for a metric."""
        metrics = self.get_agent_metrics(agent_id)
        if metric_name in metrics:
            values = metrics[metric_name]
            return sum(values) / len(values)
        return None

29.8 Best Practices

29.8.1 1. Clear Agent Responsibilities

Define clear boundaries and responsibilities for each agent. Avoid overlapping responsibilities that cause confusion.

29.8.2 2. Robust Communication

Implement reliable communication patterns with error handling and retry logic.

29.8.3 3. Monitoring and Observability

Log all agent interactions and maintain clear visibility into system behavior.

29.8.4 4. Graceful Degradation

Design systems to continue functioning even if some agents fail.

29.8.5 5. Testing

Test agents individually and as part of the overall system.

29.8.6 6. Iterative Development

Start with simple agent interactions and gradually increase complexity.

29.9 Common Pitfalls

29.9.1 1. Over-Engineering

Problem: Too many agents make the system complex and hard to debug.

Solution: Start with the minimum number of agents needed, add complexity only when necessary.

29.9.2 2. Poor Communication

Problem: Agents don’t communicate effectively, leading to misunderstandings.

Solution: Define clear communication protocols and message formats.

29.9.3 3. Deadlocks

Problem: Agents wait for each other indefinitely.

Solution: Implement timeouts and fallback mechanisms.

29.9.4 4. Inconsistent State

Problem: Agents have different views of the system state.

Solution: Use shared state management or consensus mechanisms.

29.9.5 5. Unbounded Loops

Problem: Agents enter infinite loops or never terminate.

Solution: Implement maximum iteration limits and termination conditions.

29.10 Conclusion

Multi-agent systems enable AI to tackle complex tasks through collaboration, specialization, and coordination.
